# ImageTools
  一个有关图像的工具集，暂时包括以下：
  1. Renderer: 一个基本光线追踪的渲染器，暂时实现了局部光照与全局光照（via unbiased Monte Carlo path tracing）；
  2. PXM tools: 一个简单读写 PPM / PGM / PBM 图像格式的工具；
  3. Image processing: 有关图像处理的工具，如 box blur, median blur, biliteral blur, etc.
